# Advent Accord

Durable source of truth for Advent Accord canon, visual direction, and production practice.

## Authority

1. Most-recent explicit creator decision
2. Canon documents
3. Approved visual references
4. Production documents
5. Older drafts and exploratory notes

Never silently convert a proposal into canon. Mark unresolved material **TBD**, **candidate**, or **proposal**.
